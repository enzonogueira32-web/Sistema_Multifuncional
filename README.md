
Nome do Projeto: Projeto de Funções Variadas
Objetivo

O site permite que o usuário execute pequenas tarefas interativas direto no navegador, como ordenar listas, fazer cálculos ou organizar dados, sem precisar instalar nada. É uma espécie de mini plataforma de exercícios de lógica e matemática.

Como funciona
Tela inicial – Digitar nome
O usuário digita seu nome e clica em “Começar”.
O site exibe uma mensagem de boas-vindas e pergunta se o usuário quer ver a história do projeto.
Botões Sim ou Não alteram a mensagem exibida:
Sim: conta rapidamente que o projeto foi criado para praticar lógica de programação.
Não: apenas encoraja o usuário a escolher uma função para começar.
Painel de tarefas
Depois de iniciar, aparece um painel onde o usuário pode escolher uma função no menu suspenso (select).
As funções disponíveis atualmente são:
Ordenar Chamada – Digitar nomes ou itens e o site os ordena alfabeticamente.
Ordenar Números – Digitar números e o site os ordena do menor para o maior.
Calculadora – Digitar dois números e o site calcula soma, subtração, multiplicação e divisão.
Campos dinâmicos
Para funções que pedem vários itens (como nomes ou números), o site cria automaticamente inputs para digitar cada item.
O usuário define quantos itens quer adicionar.
Para funções de campos fixos (como calculadora), aparecem inputs específicos para os números.
Executar função
Cada tarefa tem um botão Executar, que processa os dados e mostra o resultado na tela.
O resultado aparece em um quadro com fundo escuro, mantendo o estilo dark mode.
Resultados longos possuem scroll e aparecem com um efeito de fade-in, para visualização suave.
Limpar tarefa
Há um botão “Limpar Tarefa” que limpa os inputs e o resultado, permitindo que o usuário reinicie a função sem precisar recarregar a página.
