# Sistema Multitarefas

Este projeto é um sistema multitarefas desenvolvido com Python e JavaScript, capaz de executar e gerenciar várias tarefas ao mesmo tempo de forma eficiente. Ele foi criado com o objetivo de demonstrar como diferentes tecnologias podem trabalhar juntas para lidar com múltiplos processos simultâneos.

## Funcionalidades

* Execução de múltiplas tarefas simultaneamente
* Gerenciamento de processos em tempo real
* Comunicação entre backend e frontend
* Interface simples para controle das tarefas

## Tecnologias

* Backend: Python
* Frontend: JavaScript
* Comunicação: API REST

## Como funciona

O sistema permite que várias tarefas sejam iniciadas ao mesmo tempo. O backend em Python é responsável por processar e gerenciar essas tarefas, enquanto o frontend em JavaScript exibe as informações e permite a interação do usuário.

## Como executar

### Backend

```bash
python main.py
```

### Frontend

```bash
npm install
npm start
```

## Objetivo

O objetivo do projeto é demonstrar conceitos de lógica de programação, integração entre linguagens e controle de execução de processos.
